BryTorch — official pytorch suite dashboard

Runs PyTorch 2.14's own test suites (from pytorch/test/, embedded in the VFS with the real torch.testing._internal harness) against the wasm torch._C. Brython + the VFS + the wasm load once and the suites run sequentially in one interpreter. numpy (the NumBry wasm) loads alongside when present — without it the numpy-dependent suites report their import failure honestly. Per-suite detail goes to the browser console.
